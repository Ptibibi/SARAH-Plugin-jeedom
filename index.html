Le plugin jeedom permet de réaliser des interactions vocale avec la box domotique jeedom.

Il est nécessaire d'installer le plugin Sarah dans jeedom depuis le market.

Les commandes vocales sont a définir dans le menu interaction de jeedom.

Un seul plugin, deux versions de Sarah compatible

Ce plugin est compatible avec une installation standard de Sarah sur un PC Windows

ainsi que pour une version Lite de Sarah sur un Raspberry pi.

Pour plus d'information sur la version Lite de Sarah, rdv sur le wiki de jeedom

https://jeedom.fr/
